1. How to create reactive forms?

Answer:To create reactive forms in Angular is to Import ReactiveForms Module in your module file and Create a FormGroup and FormControl in your component.

2. How to add built-in validation for email in reactive form?

Answer:To add built-in email validation in a reactive form is use Validators email with FormControl.

3. How to add built-in validation for required fields in reactive form?

Answer:.To add built-in validation for required fields in a reactive form, use the Validators.required validator when defining the form control.

4. How to add regex validation for reactive forms?

Answer: To add regex validation in reactive forms, use the Validators.pattern validator when defining a form control.

5. How to generate routing?

Answer: To generate routing in an Angular application, use the Angular CLI command and This creates a dedicated routing module and integrates it with the main application module. You can then define and manage routes in the newly created module.

6. How to configure routes with parameters?

Answer: Define a route with a parameter in the RouterModule configuration, using a colon before the parameter name. And Access the parameter in the component using ActivatedRoute

7. What is DebounceTime in RxJS?

Answer: debounceTime in RxJS is an operator that delays the emission of values from an observable by a specified time duration, emitting only the latest value after the specified time if no new values are emitted during that period.

8. What is pipe operator?

Answer: The pipe operator is used in programming to pass the output of one function, command, or expression as the input to another, creating a streamlined sequence of operations is commonly used in Unix/Linux shell scripting and functional programming languages.

9. What is distinctUntilChanged operator?

Answer: The distinctUntilChanged operator in RxJS emits values only when the current value is different from the previous one, preventing duplicate consecutive emissions.

10. What is filter operator in RxJs?

Answer: In RxJS, the filter operator is used to emit only those values from an observable that satisfy a specified predicate function, effectively filtering out unwanted values.
